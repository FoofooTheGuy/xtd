# hello_world_ustring

The classic first application "Hello, World!" with [xtd::ustring](https://codedocs.xyz/gammasoft71/xtd/classxtd_1_1ustring.html) class.

## Sources

[src/hello_world_ustring.cpp](src/hello_world_ustring.cpp)

[CMakeLists.txt](CMakeLists.txt)

# Build and run

Open "Command Prompt" or "Terminal". Navigate to the folder that contains the project and type the following:

```cmake
xtdc run
```

# Output

```
Hello,     World!
```
